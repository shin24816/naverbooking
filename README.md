# 네이버 예약 자동화

하이엔소아청소년과의원 네이버 예약을 월요일 오전 9시 오픈과 동시에 자동으로 완료하는 스크립트입니다.

## 설치

```bash
pip install playwright
playwright install chromium
```

## 사용법

| 명령어 | 설명 |
|--------|------|
| `python naver_booking.py` | 실전 — 월요일 오전 9시까지 대기 후 영유아 검진 예약 |
| `python naver_booking.py --test-infant` | 영유아 검진 테스트 — 5초 로그인 확인 후 즉시 실행 |
| `python naver_booking.py --test-time 09:00:30` | 시간 지정 테스트 — 오늘 해당 시각에 영유아 검진 예약 시도 |
| `python naver_booking.py --test` | 두상 교정 테스트 — 대기 없이 즉시 실행 |

## 동작 흐름

```
서버 시간 오프셋 측정
    ↓
예약 페이지 미리 로드
    ↓
[실전] 로그인 확인 → 오전 9시 정각 대기
[--test-infant] 5초 로그인 확인 대기 → 즉시 시작
[--test] 즉시 시작
    ↓
예약하기 링크 클릭
    ↓
날짜 / 시간 선택
    ↓
[--test만] 두상 교정 체크박스 선택
    ↓
다음단계 클릭
    ↓
추가 정보 자동 입력
    ↓
동의하고 예약하기 → 완료
```

## 날짜 / 시간 선택 알고리즘

### 실전 / `--test-infant` (영유아 검진)
1. 달력에서 `class="calendar_date"` (정확히 일치) 인 날짜 중 **토요일 열(7번째 열)** 에 있는 가장 빠른 날짜 선택
2. 현재 달에 가능한 토요일 없으면 `>` 버튼으로 다음 달 이동 (최대 6달)
3. 날짜 선택 후 `class="btn_time"` (정확히 일치) 인 **첫 번째 시간** 선택

### `--test` (두상 교정)
1. 달력에서 `class="calendar_date"` (정확히 일치) 인 **가장 빠른 날짜** 선택 (요일 무관)
2. 현재 달에 없으면 `>` 버튼으로 다음 달 이동 (최대 6달)
3. 날짜 선택 후 첫 번째 시간 선택
4. **두상 교정 체크박스** 선택

> `calendar_date` 외에 `closed`, `selected`, `unselectable` 등 추가 클래스가 붙은 날짜는 예약 불가로 판단해 건너뜁니다.

## 설정값

[naver_booking.py](naver_booking.py) 상단에서 수정:

| 항목 | 변수 | 현재값 |
|------|------|--------|
| 예약 URL | `BOOKING_URL` | 하이엔소아청소년과의원 |
| 예약 목표 시간 | `TARGET_TIME` | 매주 월요일 09:00 KST |
| 아이 이름 | `CHILD_NAME` | 신서하 |
| 엄마 생일 | `MOM_BIRTHDAY` | 871019 |
| 아빠 생일 | `DAD_BIRTHDAY` | 871101 |

## 테스트 모드 상세 로그

`--test` / `--test-infant` 실행 시 달력·시간 선택 과정을 상세히 출력합니다 (실전 모드에서는 출력 안 함).

- **달을 넘길 때**: 해당 월의 예약 가능/불가 날짜 수와 불가 사유(`closed`, `unselectable` 등) 표시
  ```
  [달력] 2026년 4월: 예약가능 0일 / 불가 8일 (27일(closed selected), 28일(unselectable), ...)
  [달력] 이번 달 예약 없음 → 다음 달로 이동 (1회)
  ```
- **날짜 선택 후**: 예약 가능/불가 시간 슬롯 목록 표시
  ```
  [시간] 예약가능: 09:00, 10:00, 11:00
  [시간] 예약불가: 08:00(btn_time disabled), 14:00(btn_time closed)
  ```

## 주의사항

- 실행 전 브라우저에서 **네이버 로그인** 상태 확인 필수
- 로그인 세션은 `naver_chrome_profile/` 폴더에 저장됨
- 최종 예약까지 자동으로 진행되므로 실행 전 설정값을 꼭 확인하세요
