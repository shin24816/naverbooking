# 네이버 예약 자동화

하이엔소아청소년과의원 네이버 예약을 월요일 오전 9시 오픈과 동시에 자동으로 완료하는 스크립트입니다.

## 브랜치

| 브랜치 | 용도 |
|--------|------|
| `master` | 영유아 검진 (다음주 토요일) 실전 예약 |
| `test/dusang-booking` | 두상 교정 예약 테스트 |

## 설치

```bash
pip install playwright
playwright install chromium
```

## 사용법

### 실전 예약 (월요일 오전 9시)
```bash
python naver_booking.py
```

### 테스트 (대기 없이 즉시 실행)
```bash
python naver_booking.py --test
```

## 동작 흐름

```
서버 시간 오프셋 측정
    ↓
예약 페이지 미리 로드 (로그인 확인)
    ↓
오전 9시 정각 → 예약하기 링크 클릭
    ↓
날짜 / 시간 선택 (아래 알고리즘 참고)
    ↓
다음단계 클릭
    ↓
추가 정보 자동 입력 (동의 항목 / 아이 이름 / 부모 생일)
    ↓
동의하고 예약하기 → 예약 완료
```

## 날짜 / 시간 선택 알고리즘

### master (영유아 검진 — 토요일)
1. 달력에서 `class="calendar_date"` (정확히 일치, 다른 클래스 없음) 인 날짜 중 **다음 주 토요일** 선택
   - 오늘 기준 `(6 - 요일 + 7) % 7 + 7` 일 후의 날짜를 계산
   - 해당 날짜가 달력의 **7번째 열(토요일 열)** 에 있는지 확인 후 클릭
2. 날짜 선택 후 나타나는 시간 목록에서 `class="btn_time"` (정확히 일치) 인 **첫 번째 시간** 선택

### test/dusang-booking (두상 교정)
1. 달력에서 `class="calendar_date"` (정확히 일치) 인 **가장 빠른 날짜** 선택
   - 현재 달에 없으면 `>` 버튼으로 다음 달 이동 (최대 6달)
2. 날짜 선택 후 나타나는 시간 목록에서 `class="btn_time"` (정확히 일치) 인 **첫 번째 시간** 선택
3. **두상 교정 체크박스** (`data-click-code="options.checkbookingcount0"`) 선택

> `calendar_date` 외에 `closed`, `selected`, `unselectable` 등 추가 클래스가 붙은 날짜는 예약 불가로 판단해 건너뜁니다.  
> `btn_time` 외에 추가 클래스가 붙은 시간대도 마찬가지로 건너뜁니다.

## 설정값

[naver_booking.py](naver_booking.py) 상단에서 수정:

| 항목 | 변수 | 현재값 |
|------|------|--------|
| 예약 URL | `BOOKING_URL` | 하이엔소아청소년과의원 |
| 예약 목표 시간 | `TARGET_TIME` | 매주 월요일 09:00 KST |
| 아이 이름 | `CHILD_NAME` | 신서하 |
| 엄마 생일 | `MOM_BIRTHDAY` | 871019 |
| 아빠 생일 | `DAD_BIRTHDAY` | 871101 |

## 주의사항

- 실행 전 브라우저에서 **네이버 로그인** 상태 확인 필수
- 로그인 세션은 `naver_chrome_profile/` 폴더에 저장됨
- 최종 예약까지 자동으로 진행되므로 실행 전 설정값을 꼭 확인하세요
